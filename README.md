# lyGrid document

## 说明

这是一个基于 Bootstrap 抽取的栅格样式文件

## 目的

在前端开发场景中，我们很多时候会用到芬栏布局，其中 Bootstrap 就为我们提供了这样一个栅格系统，方便我们快速的进行页面布局。
但在某一些项目中，我们可能只需要用到 Bootstrp 的栅格系统，这个时候如果引用整个 Bootstrap 的样式文件，会增加前端工程的项目提及，因此我单独抽离出了 lyGird.css，解决了这一个环节的痛点！
